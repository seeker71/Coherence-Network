# Changelog

Release notes for each merge to `main` are published as [GitHub Releases](https://github.com/seeker71/Coherence-Network/releases). Release bodies are generated from Conventional Commits using `scripts/changelog_from_conventional.py` (see `.github/workflows/auto-publish-main.yml`).

This file is the canonical pointer for changelog policy; per-release Markdown is attached to the GitHub Release for that push.
