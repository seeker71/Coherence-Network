# BML — Backtracking Model Language

*Work · 2000 · Master's Thesis*

The language layer of the 2000 Bjorg-Muff thesis at CU Boulder. Synthesizes **Prolog**'s operational semantics, **Smalltalk-80**'s object/image model, **Java**'s typed garbage-collected runtime, and **C++**'s component-host pragmatics — all four ancestors arranged at four altitudes of the same building. BML is self-describing: its parser grammar ( [BMF](/people/bmf-grammar) ) is itself written in BML.

## What BML — Backtracking Model Language has given the Coherence Network

The thesis itself is 11,566 words; the conclusion was left as three subheadings without body. The work was *delivered* — defense, photographs, the running VM — but the prose summary stayed open. This page is the twenty-six-years-later attempt to close that breath: the substance the conclusion never wrote, in this medium, with the diagrams a 2000 Word document couldn't carry.
