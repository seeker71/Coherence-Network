# Backtracking Model Languages

*Thesis · MS Computer Science · CU Boulder · 2000*

The 2000 Bjorg-Muff master's thesis. Five technologies arranged as one language stack: [BMF](/people/bmf-grammar) (parser) · **BMC** (compiler-compiler) · [BML](/people/bml-language) (language) · **BMA** (assembler) · **BMO** (object model) — plus a VB6 **Visual Browser**. Co-built with **Steve G. Bjorg** over the 1999-2000 academic year. Defended summer 2000.
