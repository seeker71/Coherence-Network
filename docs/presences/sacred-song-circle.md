# Sacred Song Circle

An international network of kirtan teachers tending one shared website that lists biographies, songbooks, and touring schedules for devotional musicians in the bhakti and ecstatic-singing field. The site is where [Vasudev Baba](/people/vasudev-baba)'s public bio is tended, alongside other teachers in the lineage of sacred-name singing. The network operates without a single named founder surfaced — a teacher-cooperative shape rather than a school.

**What Sacred Song Circle has given the Coherence Network.** The public anchor for Vasudev Baba's biographical arc — the source of canonical lineage detail (Oslo, 1987, Indian Himalayas, Brazilian jungles, Rainbow Gatherings). Teacher-cooperative shape (no single named founder; many teachers tending one shared site) is resonant with [lc-sovereignty-within-oneness](/vision/lc-sovereignty-within-oneness) (many sovereign cells, one organism). The bhakti lineage of singing the Divine Names is one of the body's anchor frequencies for the arrival-as-frequency framing — see [lc-frequency-routes-reception](/vision/lc-frequency-routes-reception).

Public anchors: [sacredsongcircle.com](https://www.sacredsongcircle.com/) · [Vasudev profile](https://www.sacredsongcircle.com/vasudev/) · [Testimonials](https://www.sacredsongcircle.com/testimonials/) · [Facebook group](https://www.facebook.com/groups/123972024418527/).
