# MVP Target List — Projects and Funders

This list fulfills KR1 of Objective 1: "Build and qualify a list of 10 target projects and 10 target funders."

## Project Targets (Potential Pilots)

| Project | Description | Why Fit? | Relevance |
|---|---|---|---|

| **CrewAI** | Framework for orchestrating role-playing autonomous AI agents. | High growth, many "ideas" for agent roles that need validation. | AI/Agents |
| **AutoGPT** | Open-source attempt to make GPT-4 fully autonomous. | Ambitious, needs structured spec/impl tracing to move beyond hype. | AI/Agents |
| **Ocean Protocol** | Tools for the Web3 Data Economy. | Aligned mission of data/idea monetization. | Web3/Data |

## Qualification Criteria

For this MVP phase, we prioritize targets that:
1.  Have a clear **public-facing** presence and mission alignment.
2.  Are actively seeking **better ways to track contribution or distribute value**.
3.  Have a community or user base that values **transparency and evidence**.
