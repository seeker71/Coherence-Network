## Current Focus

`NEXT ROI ITEM (in progress)`: Craft mission and scope.

### Mission (v1)

Build a trusted marketplace that connects aligned investors/funders with pilot projects, while running a conservative income sleeve that extends runway during build-out.

### Scope (v1)

In scope for NVP phase:
- Attract and qualify 3-5 pilot projects.
- Attract and qualify initial funders/investors.
- Track each project from intake -> evaluation -> commitment -> execution readiness.
- Operate a documented quality/trust process for onboarding and ongoing participation.
- Maintain a simple investment allocation log and monthly review cadence.

Out of scope for NVP phase:
- Full-scale automated trading/investment infrastructure.
- Complex compliance automation or custom legal tooling.
- Enterprise-grade multi-tenant marketplace feature set.

Success definition for this phase:
- 3-5 pilot projects actively tracked.
- At least 2 funding counterparties in active conversation.
- One repeatable onboarding checklist used across all pilots.
- Weekly progress dashboard reviewed with explicit next actions.

Objective 2: Operationalize trust and execution basics.
- KR1: Finalize one-page AI security and approval policy for external-facing actions.
- KR2: Publish a pilot onboarding checklist and quality gate used for every intake.
- KR3: Stand up weekly dashboard review with owner, date, and explicit next actions logged.

Objective 3: Build financial discipline while MVP planning progresses.
- KR1: Publish allocation policy with target ranges (runway, yield sleeve, operating spend).
- KR2: Execute first allocation action only after policy sign-off and record rationale.
- KR3: Run one weekly variance review comparing planned vs actual capital allocation.
