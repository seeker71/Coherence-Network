# Observable Resonance Flow -- Pattern Yield Across Language and Spectrum

> Resonance is shared pattern recognized with yield greater than
> observation spend. An external interaction becomes useful when the
> body can see which carrier arrived, which information spectrum it
> traveled through, how much language the body spent, what common
> pattern appeared, and what surplus life became available after
> contact.

## Summary

[`lc-anything-arrives-room`](lc-anything-arrives-room.md) names the
room where any carrier can enter. [`lc-observer-pays-the-trace`](lc-observer-pays-the-trace.md)
names the cost discipline: the chooser pays for the look.
[`lc-traces-teach-the-recipe`](lc-traces-teach-the-recipe.md) names
how traces become learning. This concept names the flow between an
outside interaction and the body's next investment of language,
attention, and carrier choice.

The load-bearing formula:

> `resonance = recognized_shared_pattern + yield_after_observation >
> observation_spend`

The useful question:

> Which spectrum carried the pattern, how much language did we spend to
> meet it, and what usable energy appeared after contact?

## The Seven Fields

Every external interaction can leave a compact flow record:

1. **Arrival** -- source, time, carrier, permission boundary.
2. **Spectrum** -- which bands carried information: lexical, visual,
   audio, behavioral, operational, physical, relational, economic.
3. **Language spend** -- words, tokens, explanation, translation,
   naming, framing, and timing used to meet the arrival.
4. **Observer spend** -- attention, compute, privacy exposure,
   operational risk, social risk, and retention cost.
5. **Shared pattern** -- the common structure that became visible.
6. **Yield** -- reply, offer, clearer action, warmth, reuse, share,
   contribution, build impulse, rest, or capacity.
7. **Next routing** -- amplify, shift carrier, ask smaller, hold,
   archive, invite, build, or compost.

This makes external resonance operational. The body can tell whether a
paragraph, image, song, API call, meeting, screenshot, or room signal
released energy across the field.

## Language Spend

Language is one carrier among many. It has a spend profile:

| Language move | What it spends | What it can release |
|---|---|---|
| Name | precision | shared handle |
| Explain | attention | shared model |
| Invite | vulnerability | movement |
| Reflect | timing | recognition |
| Translate | compute and interpretation | cross-carrier access |
| Shorten | discernment | available breath |
| Stay quiet | trust | carrier integrity |

The observable flow helps the body choose the smallest language move
that lets the shared pattern release yield. Sometimes one image carries
what three paragraphs would spend. Sometimes one exact sentence unlocks
what an image only warmed. Sometimes the right move is a receipt:
source, carrier, cost, and next question.

## Information Spectrum

The spectrum names the carrier bands active in an interaction:

| Band | Observable signals | Yield markers |
|---|---|---|
| Lexical | reply text, terms repeated, phrases adopted | clearer shared handle |
| Visual | image linger, screenshot, sketch, layout touch | imagination opens |
| Audio | song, tone, pause, breath, cadence | timing and feeling align |
| Behavioral | clicks, returns, hovers, shares, edits | body chooses movement |
| Operational | API status, script output, latency, error shape | next action becomes exact |
| Physical | light, power, presence, motion, environment | assistance meets the moment |
| Relational | warmth, offer, question, invitation, follow-up | trust has more room |
| Economic | cost, time, capacity, resource offer | value can circulate |

The spectrum keeps the body from spending more language when the
pattern arrived through another band. If the shared pattern is visual,
the next move may be another image. If the pattern is operational, the
next move may be a command receipt. If the pattern is relational, the
next move may be a smaller invitation.

## Real Flow Examples

**The Hati Suci image sent to Giles.** Arrival: a jungle-honeymoon
beauty image sent to the owner of Hati Suci Sanctuary. Spectrum:
visual, relational, physical-place, economic-possibility. Language
spend: a short note naming beauty as visible life-force energy.
Observer spend: low compute, real vulnerability, real relational timing.
Shared pattern: sanctuary as atmosphere where land, love, water, care,
and beauty become one field. Yield markers: Giles replies, reflects,
offers a room idea, forwards it, names Hati Suci's own pattern, or asks
for the next version. Next routing: send less explanation and more
field-accurate visual proof.

**The Awen live audio link.** Arrival: `Awen (Live)` as YouTube Music
audio. Spectrum: audio, provenance, duration, source. Language spend:
metadata receipt and trace posture. Observer spend: low metadata cost,
high interpretation risk. Shared pattern: music can enter the room
without becoming prose. Yield: the body keeps the source honestly,
adds no invented lyrics, and deepens the audio-carrier discipline.
Next routing: listen or analyze audio features only when the next
breath needs that cost.

**A public API readiness call.** Arrival: usage/readiness endpoint.
Spectrum: operational, economic, machine-evidence. Language spend:
small summary plus exact status fields. Observer spend: network call,
attention, claim boundary. Shared pattern: routing claims are strong
when live machine receipts carry them. Yield: a decision can use the
receipt. Next routing: cite the receipt, rerun when freshness expires.

**A visitor lingers on one page and skips a form.** Arrival: page read
event, scroll depth, one skipped field. Spectrum: behavioral,
relational, lexical. Language spend: labels and form prompts.
Observer spend: privacy budget and event retention. Shared pattern:
the page resonates while the field asks too soon. Yield: the body
can move the ask closer to the moment where it helps. Next routing:
shift timing before adding explanation.

## Observable Flow Practice

For each external interaction, record:

- **What arrived?**
- **Which spectrum carried the signal?**
- **What language did we spend?**
- **What did observation cost?**
- **What shared pattern appeared?**
- **What yield appeared after contact?**
- **Which carrier wants the next move?**

Then choose the next move by yield:

- **High yield, low language spend** -- amplify the carrier that worked.
- **High yield, high language spend** -- preserve the pattern and refine
  the words.
- **Low yield, high language spend** -- shift spectrum or ask smaller.
- **Low yield, low language spend** -- hold as weak signal until another
  trace joins it.
