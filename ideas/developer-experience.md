# Developer Experience

Make contributing to the platform fast and frictionless.

## What It Does

- Local CI parity: tests run the same locally and in CI
- Worktree-based workflow: isolated branches for parallel development
- Documentation hygiene: clear runbook, setup guide, API docs
- Test suite: 177 flow-centric tests in 8s

## Why It Matters

Contributors are the platform's lifeblood. If contributing is painful, nobody will.
