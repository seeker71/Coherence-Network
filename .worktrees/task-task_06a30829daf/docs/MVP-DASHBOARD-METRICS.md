# MVP Dashboard Metrics (Weekly)

Last updated: 2026-03-11
Owner: `seeker71`
Cadence: weekly (same day/time each week)

## Purpose

Track whether the MVP marketplace is moving toward validated demand, trusted execution, and funding readiness.

## Core Weekly Metrics

### Demand and pipeline
- `targets_total`: total active project + funder targets.
- `outreach_sent_week`: first-touch messages sent this week.
- `reply_rate_week`: replies / outreach sent (week).
- `discovery_calls_week`: completed calls this week.

### Pilot readiness
- `pilot_projects_active`: projects in active pilot funnel.
- `funders_active`: active funder/investor conversations.
- `qualified_matches`: project-funder pairs with clear potential fit.

### Trust and quality
- `onboarding_checklist_usage`: % of new intakes using checklist.
- `quality_review_pass_rate`: pass / total reviewed this week.
- `security_policy_compliance`: compliant actions / total external-facing actions.

### Financial discipline
- `allocated_capital_total_usd`: total capital allocated under policy.
- `yield_sleeve_allocated_usd`: amount in income sleeve.
- `policy_variance_count`: number of allocation actions outside policy guardrails.

## Review Process

1. Update metrics before weekly review.
2. Compare against 30-day OKRs in `docs/MVP-MARKETPLACE-STATUS.md`.
3. Record:
- 3 wins
- 3 blockers
- 3 explicit next actions with owner/date

## Minimum Weekly Standard

- Outreach sent this week > 0
- At least 1 discovery call or confirmed no-fit decision
- Dashboard row completed with next actions
