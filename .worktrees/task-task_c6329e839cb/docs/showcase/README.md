# From Idea to Reality — Three Stories

Every idea deserves to become real. These three stories show how the Coherence Network takes ideas — not just software ideas, *any* ideas — from a spark in someone's head to measurable impact in the world.

Each story follows the full chain:
**Idea → Questions → Spec → Work → Measurement → Proof of Impact**

Each story is backed by actual API calls you can run right now.

## The Stories

1. **[The Seed Library](01-seed-library.md)** — A retired teacher starts a community seed library. Fifteen neighbors join. By spring, 200 seed packets circulate across 3 neighborhoods. The platform tracked every contribution, every question answered, every volunteer hour — and proved that $47 in materials generated $2,000 in garden produce.

2. **[The Accessibility Extension](02-accessibility-extension.md)** — A blind developer proposes a browser extension that reads page structure aloud. A designer, two testers, and a student join. In 6 weeks the extension has 340 users. The platform measured which features actually helped and which were noise.

3. **[The School Lunch Revolution](03-school-lunch.md)** — A parent notices kids throwing away cafeteria food. She proposes a menu feedback system. The school board resists. The platform's governance flow lets 40 parents vote, the data proves the case, and the district changes the menu. Waste drops 35%.

## Use It From Any AI

These stories aren't locked to one tool. You can interact with Coherence Network from:

- **ChatGPT** → [Custom GPT Setup Guide](../integration/chatgpt-setup.md)
- **Grok** → [Grok Integration Guide](../integration/grok-setup.md)

- **Cursor / Windsurf** → MCP config (same server)
- **Any HTTP client** → REST API at `https://api.coherencycoin.com/docs`
- **Terminal** → `cc` CLI (`pip install coherence-cli`)
