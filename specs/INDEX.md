## What We Are NOT Building Yet

- **Social platform bots** -- no user base to broadcast to
- **Belief systems / worldview translation** -- philosophical, not practical for MVP
- **Fractal navigation / resonance navigation** -- UX polish before core works
- **Crypto treasury bridge** -- no users to transact with
- **Marketplace** -- no federation to trade across
- **Garden metaphor UIs** -- visual polish before data model is proven

## Roadmap

### Phase 1 (NOW): Core Idea Lifecycle + Pipeline Reliability

- Ideas API with prioritization, lifecycle, and closure
- Contributions tracking and value lineage
- Coherence Credit as unit of account
- Agent pipeline: reliable task execution, diagnostics, auto-heal
- CLI and web: functional surfaces for idea and pipeline management
- PostgreSQL migration for production persistence

### Phase 2 (NEXT): Multi-User + Federation

- Contributor onboarding and identity (TOFU)
- Federation layer: node identity, sync, aggregated visibility
- OpenClaw marketplace for cross-instance idea trading
- Investment UX: stake CC on ideas

### Phase 3 (LATER): Advanced Intelligence + Ecosystem

- Resonance navigation and fractal zoom
- Social platform integrations
- Crypto treasury bridge
- Belief systems and worldview translation
- Community funder matching
